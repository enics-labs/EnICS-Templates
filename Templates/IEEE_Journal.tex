    \begin{itemize}
        \item The \textbf{EnICS Templates} project is general covering many different types of templates. 
              You only need this one, so \textbf{copy} this file into the root directory and erase the \texttt{main.tex} file and all the template files you do not need (actually, the entire \texttt{Templates} folder).
        \item If it wasn't clear, \textbf{erase} the \texttt{\textbackslash ExecuteMetaData} commands. 
              These are just to put some explanations for you (defined in the \texttt{templateText.tex} file) 
              and you should fill in your own text instead.
        \item Put your paper-specific acronyms and newcommands inside the \texttt{this\_glossary.tex} file.
        \item Put your paper-specific references inside the \texttt{this\_bibliography.bib} file.
    \end{itemize}
